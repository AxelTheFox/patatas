Símbolos y caracteres especiales

Ejemplos de uso

En este documento, se muestran algunos ejemplos de cómo utilizar símbolos HTML y caracteres especiales en el código HTML.

Símbolos de ángulo (< y >)

Los símbolos de águlo (< y >) se utilizan para indicar el **inicio** y el **final** de las etiquetas HTML. Por ejemplo:

<p>Este es un párrafo de texto </p>

Símbolo de entidad HTML (&)

El símbolo de entidad HTML (&) se utiliza para representar el carácter especial ampersand (&). Por ejemplo:

Esta frase contiene un &ampersand

Símbolos de moneda

HTML incluye símbolos para diferentes monedas, como la libra (£), el dólar ($), el euro (€), el yen (¥) y otros.

  • Precio en libras: £10.99
  • Precio en dólares: $19.95
  • Precio en euros: €24.50
  • Precio en yenes: ¥2,800

    Comillas

    HTML permite utilizar diferentes tipos de comillas para delimitar el texto, como las comillas simples ('), comillas dobles (") y comillas angulares (”).

    Ejemplo con comillas simples: 'Este es un ejemplo de texto entre comillas simples'.

    Ejemplo con comillas dobles: "Este es un ejemplo de texto con comillas dobles".

    Ejemplo con comillas angulares: ”Este es un ejemplo de texto entre comillas angulares”.
